Задача: создать интерактивную мини-программу «Проверка заказов кафе»
    1. Создать набор переменных:
        o Название кафе (name)
        o Средний чек (averageBill)
        o Признак работы доставки (hasDelivery - true/false)
        o Вывести информацию о кафе в консоль одной строкой
        (Пример: “Кафе Ваниль средний чек 500₽ доставка: есть”)
    2. Работа с условиями:
        o Если averageBill меньше 500 вывести «Дешево и сердито»
        o Если от 500 до 1000 вывести «Средний уровень»
        o Если больше 1000 вывести «Премиум кафе»
    3. Создать массив и использовать цикл:
        o Создать массив orders = [120, 560, 310, 800, 250]
        o С помощью цикла for вычислить сумму всех заказов больше 300
        o Вывести результат в консоль
    4. Работа с методами массивов:
        o Создать массив menu = ["кофе", "чай", "круассан", "сэндвич"]
        o Добавить в конец "десерт" и удалить первый элемент
    5. Создание и использование функции:
        o Написать функцию calcDiscount(price), возвращающую цену со скидкой 15%
        o Проверить работу функции, вызвав её для разных значений
